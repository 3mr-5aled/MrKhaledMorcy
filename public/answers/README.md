# Answers Directory

This directory stores uploaded answer files organized by grade and unit:

```
answers/
├── prep-1/
│   ├── unit-1/
│   ├── unit-2/
│   └── ...
├── prep-2/
├── prep-3/
├── sec-1/
├── sec-2/
└── sec-3/
```

Files are automatically created when answers are uploaded through the admin dashboard.
